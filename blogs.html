Why you should switch to Linux

Linux is a kernel, but popularly reffered to as an operating system. A lot of open source operating systems are based on GNU + Linux these operating systems are famous for their stability, and ironically, low marketshare.

What a lot of people don't know is that Android which is the most popular OS runs on the linux kernel, so do a lot of servers from big companies. There is a high chance that the windows iso download is from a linux server.

Stability:
Linux based operating systems are highly stable, and optimized, as they are open source, there is very little amount of bloatware in the OS. Unlike Windows, these operating systems or distributions as they are called do not share your personal data as telemetry, which is why even very old hardware can run linux easily, even the Rasberry Pi can runs a linux distribution. A huge reason why Linux is used for servers is because it can update in realtime without any need to restart, it is highly stable and light, and can be modified to any level by the companies that use them, and the distributions are free of cost. Unlike other operating systems, the linux distros can update their entire system without needing to restart, while windows has to restart everytime you install a new program, such that it works properly. Even Azure, microsoft's cloud computing services use linux based servers.

Security:
Linux is hugely known for it's security. Unlike Windows, linux has a better system for administrator access, so a lot of times you don't need it to do what you want, and if you do, you always need to put in the root password, while in windows you have to make a new userprofile just for admin purposes. Many distros come with a package manager, which makes installing programs a lot easier, you do not have to go to the website and download executable binaries to install programs, you just have to type a sentence like "sudo apt install brave" in the terminal and it will install the program "brave" using the apt package manager with root(sudo) permissions, these programs are stored in secure repositories so this way installing malware even by mistake is quite difficult. Updating software is thus a lot more easier as it can be done with just one command, "sudo apt update". Because linux distros are used lesser, and by more experienced users, the amount of malware made for it is also much lesser.

Customizability:
Linux distributions are very highly customisable, a user can make their user interface look like whatever they want, it can look exactly like windows or, MacOs, or just their own custom UI. You can change desktop environments which basically changes how your UI looks, you can edit any UI element and modify it to your liking, without installing any extra programs. You can run custom shell scripts to do automated tasks such as download songs from a playlist on youtube to a specific folder. You can install libraries and scripts to make your PC look like whatever you want, the open source community makes a huge amount of custom themes, color schemes, Programs and they can be installed and used without any problem instantly. It is quite amazing how a user can do whatever they wish to do, many people start "ricing" or adding custom code to their configurations to suit their needs.

In conclusion, a lot of people can and should use Linux based distributions as their operating systems, all it takes is a few minutes and a USB stick. If you wanna try out linux, you should start with a quite popular distribution Ubuntu.

Why everyone should use FOSS

If you clicked on this then you probably know what Free and Open Source Sofware or FOSS is and how amazing,customisable and community supported it is, if you don’t let me explain.

FOSS means Free and Open Source Software.Open source basically means that the code running the software is publicly available, anyone can modify it to their own liking or if authorised can change the official code to increase its functionality. Software used by the majority of the public is closed source, let’s talk about a popularly used piece of software, Adobe Photoshop. The code that makes it work is only known by Adobe, which means that if someone finds a bug in the software they can’t find the code and fix the bug, they’ll instead have to report the bug to Adobe,and they might not even fix the bug which brings us to the first problem posed by closed source software, that is people can’t find and fix bugs on their own, everything is done by the company making the software and because there are only a limited amount of employes working at the company sometimes there are many bugs that aren’t fixed or even known, and even a person who could potentially fix it won’t be able to because the software can’t be modified. If vunerabilites or bugs are found in Open source software they are fixed quickly.

Another problem with closed source software is that mostly it’s paid and it’s costly, too costly .PHOTOSHOP COSTS 1675Rs PER MONTH. Which is why many people pirate closed source software for their devices which is illegal and immoral, and pirating software can also be highly unsafe.A valid point that can be raised is if people know how the code works, can’t they easily find vulnerabilities and exploit them. That doesn’t happen, there’s more good people than bad people, the software is continuously analysed and viewed by many people, hobbists, dedicated coders working to improve, people who just wanna mess around for fun, all of them can contribute to the software making it more secure, thus vunerabilites don't last for long and are found very quickly.

When closed source software isn't paid, it's a bigger problem, because they need some way to make money, if you the aren't then well you are the product. There's a popular and a very inaccurate quote "If you're not paying for the product, you're the product" Now that might be correct for closed source platforms like Meta, and Google, that use data and anyalytics to track the users for advertisements. A lot of times in these services, you have very little privacy, they have backdoors to all your private texts with other people, they track every post you like, how much time you spend looking at a post. A lot of people don't know that because while signing up, they agree with the terms and conditions without reading them. A company like meta has a very flexible ToS which allows them to take your data and use it to their liking. Open source software unlike that, cannot and will not do that, because it's open source and transparent. In Windows 10, there are advertisements in the start menu, it is filled with unneccesary bloatware and the telemetry continously sends data back and forth from their servers thus using precious resources and the users cannot know what Microsoft is doing with that data.

It's amazing software, that doesn't get enough attention, because it isn't advertised, it doesn't need to be, it's just made, people who want to use it can use it, the only way most of the organisations get money is by donations. Awesome Applications

I’ve used Android a lot, and through the years I’ve found many applications that make my life a lot easier. Some of these even work on windows and linux. I wanted to share these applications with their pros and sometimes cons so here’s a list of those apps.

Droid-ify

This is the most important application of the list. Droid-ify is an open source store which downloads and automatically updates applicaions from many open source repositories such as F-Droid and IzzyOnDroid. Basically all the apps below won't be avaliable on the play store, also will not have proprietary google API, but will definitely be here and will automatically to the latest update.Choose the repositories F-Droid and IzzyOnDroid to get started.

Stremio

If you’ve ever wanted to watch Movies or TV shows on your devices then you should definitely check out Stremio. Stremio has the ability to directly find and stream torrents effortlessly, so you can enjoy whatever content you want. Using this add-on.

Saikou

Stremio has anime, but if you want an amazing UI , Manga and the ability to update your anilist then you should check out Saikou. It scrapes shows and movies from various databases such as GOGO and tenshi.moe and even YouTube. And the Video Player for this has a lot of features too.

xManager

TL;DR Spotify on android without any advertisements.You must have noticed all the extremely annoying advertisements in spotify. You can buy spotify premium and get rid of them. But if you don’t want to spend your money on a multimillion dollar company each month. You can get spotify without any ads with xManager.

BlackHole

If you still wanna stream music, but using a completely open source application, that allows you to stream high quality music and download it in the .m4a format. BlackHole is a great choice. It has a nice and customizable UI and you can import your youtube or spotify albums, and download them as music files of high quality.

Aliucord

Discord is a pretty good social media platform, but the android app lacks customization, which is why Aliucord exists. It has the ability to apply custom themes, fonts and various plugins. It’s basically like betterdiscord for android.

Instander

Instander is a modded app for instagram on android, which removes advertisements. It has features like ghost mode and HQ media processing, downloader. It enables you to view stories and Read messages without anyone knowing.

Infinity

Reddit, the android app for Reddit sucks. Videos don’t work,the app keeps crashing, has a lot of bugs, looks ugly, cluttered interface and a lot of ads. Infinity for reddit fixes all of that, and it comes with amazing UI and customizability. The best way to experience reddit on your phone.

Metro

Metro is a fork of Retro Music, an offline music player with a beautiful interface and amazing features. It gets album and artist data from the internet and integrates it within the application. And is very highly customizable.(uses Material 3 guidelines).

NewPipe

Since youtube vanced is now deprecated the only way to enjoy content from youtube without ads is using NewPipe. It offers many features without tracking you and getting rid of advertisements. It might not have the best interface, but it’s better than watching Adverts.

LibreTorrent

If you’ve ever wanted to download Torrents on the go there is no better application than Libre Torrent, I think. It’s open source (ofc) , it has a nice UI and a lot of features, it supports BitTorrent(2.0) and web torrent, and you can stream the files using sequential downloads.

Signal

WhatsApp doesn’t have a good track record. There have been various chat leaks, privacy concerns which makes the service unreliable. Signal is open source has trusted end-to-end encryption and has more features than WhatsApp. Supports file sharing up to 100Mb, the user doesn’t have to download all the data, saving storage space.

Tachiyomi

Saikou has manga, but if you want to read manga on the go, then tachiyomi is a great app. It follows MD3 guidelines, has a lot of different sources, and you can download different chapters, continue where you left off, get updates on new chapters, it offers many settings so you can customise your experience with the app.
